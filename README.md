###### **CHAD Shoes Management System**



###### **OVERVIEW**

CHAD Shoes is a comprehensive database management system designed for a shoe manufacturing unit. It helps manage raw materials, human resources, produced items, rejected materials, and dispatch logistics. The program improves efficiency, accountability, and decision-making by organizing all data effectively.



###### **FEATURES**

* Raw Material Management: Stores information about raw materials including ID, name, quantity, and supplier. Helps maintain inventory and avoid production delays.
* Human Resource Management: Maintains employee records, including ID, name, associated materials, and salary. Simplifies payroll, performance tracking, and staffing.
* Produced Material Tracking: Keeps track of finished products with details like product ID, name, raw material used, price, and quantity produced. Helps manage inventory and sales.
* Rejected Material Logging: Records defective items along with quantity and reasons. Supports quality control and reduces waste.
* Dispatch Management: Manages the delivery of products to different locations with details like quantity, dispatch code, and destination. Ensures timely and accurate delivery.
* Data Organization \& Scalability: Well-structured tables make data easy to manage and analyze. The system can grow with the business by adding new features or tables.



###### TECHNOLOGIES / TOOLS USED

* Relational Database Management System (MySQL, PostgreSQL, etc.)
* SQL for database operations
* Optional interface in Python, Java, or web-based UI

* Create a database for CHAD Shoes in your chosen DBMS.
* Create the following tables: Raw\_Material, Human\_Resource, Produced\_Material, Rejected\_Material, Dispatch.
* Insert any initial data if available.
* Run the program or interface to manage production, inventory, and dispatch.
